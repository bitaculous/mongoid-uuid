**Universally unique identifier (UUID) for [Mongoid] documents**

Development
-----------

### Run Specs

Run `rspec`.

or via [Guard]:

```
$ guard -g spec
```

### See Test Coverage

Run `COVERAGE=true rspec`.

### Run [RuboCop]

Run `rubocop`.

To run Specs and RuboCop altogether, run `rake`.

Bug Reports
-----------

and submit new problems [here].
