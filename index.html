Mongoid-uuid

Universally unique identifier (UUID) for Mongoid documents

View the Project on GitHub bitaculous/mongoid-uuid

  • Download ZIP File
  • Download TAR Ball
  • View On GitHub

Usage

class Sample
  include Mongoid::Document
  include Mongoid::Uuid

  field :name
end

sample = Sample.create! name: 'Condimentum Ligula'

sample.uuid # Returns a `BSON::Binary` object
sample.uuid.data # Returns UUID as string

Development

Run Specs

Run rspec.

or via Guard:

$ guard -g spec

See Test Coverage

Run COVERAGE=true rspec.

Run RuboCop

Run rubocop.

To run Specs and RuboCop altogether, run rake.
